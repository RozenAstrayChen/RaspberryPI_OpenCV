# RaspberryPI_OpenCV
- Use xcode as an editor, and then throw the project to scaspberry pi through scp.
- if you want build...
```
# sudo cmake . 
# sudo make
# ./main
```
- if you want cmake to Xcode project
```
cmake -GXcode ..
```
